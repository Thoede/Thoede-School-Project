Web Profile Konsentrasi Keahlian DPIB
==========================================

Deskripsi Proyek:
Proyek ini adalah platform berbasis web untuk mengelola profil siswa skensa dalam jurusan DPIB (Desain Pemodelan Informasi Bangunan). Website ini memungkinkan pengguna untuk melakukan login, registrasi, melihat daftar kelas, dan mengakses profil siswa secara detail. Proyek ini dirancang untuk memfasilitasi pengelolaan dan tampilan informasi bagi mahasiswa yang mengikuti konsentrasi dalam pemodelan dan desain bangunan.

Fitur:
1. Halaman Login
   - Memungkinkan pengguna untuk masuk menggunakan email dan password.
   - Mendukung otentikasi untuk mengakses halaman-halaman lain.

2. Halaman Register
   - Menyediakan formulir registrasi bagi pengguna baru untuk membuat akun.
   - Membutuhkan username, email, dan password untuk registrasi.

3. Home Profile Konsentrasi Keahlian
   - Menampilkan informasi umum tentang konsentrasi mahasiswa di program DPIB.
   - Memberikan halaman sambutan untuk memperkenalkan pengguna pada program tersebut.

4. Daftar Kelas dan Siswa
   - Menampilkan daftar kelas yang tersedia dan siswa yang terdaftar.
   - Menampilkan informasi mengenai berbagai kelas dalam program tersebut.

5. Detail Siswa
   - Menampilkan informasi detail mengenai seorang siswa tertentu, termasuk mata kuliah, nilai, dan detail relevan lainnya.

Bahasa Pemrograman yang Digunakan:
- HTML: Untuk membuat struktur halaman web.
- CSS: Untuk styling halaman web dan membuatnya responsif.
- JavaScript: Untuk interaksi dinamis dan validasi formulir.


Penggunaan:
1. Halaman login untuk masuk atau registrasi akun baru.
2. Setelah login, Anda dapat mengakses halaman Home Profile untuk melihat   profil jurusan.
3. Arahkan ke Daftar Kelas dan Siswa untuk melihat daftar kelas yang tersedia.
4. Klik pada siswa manapun untuk melihat profil detail siswa tersebut.

Lisensi:
Proyek ini bersifat open-source dan telah mendapatkan izin dari jurusan yang bersangkutan.
